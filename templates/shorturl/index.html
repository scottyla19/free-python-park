URL Shortener API

Add a valid url after this url with new/ prepended. For example {{request.build_absolute_uri}}/new/http://www.google.com. You will recieve a response in JSON that shows your original url and the 'shortened' url. Try to access the shortened url and get redirected. Totaly cool, even if not really practical or shortened at all.

Example:

{{request.build_absolute_uri}}new/http://www.google.com
{"original": "http://www.google.com<", "short": "{{request.build_absolute_uri}}Amgtx2"}

You could then access {{request.build_absolute_uri}}Amgtx2 and be redirected to Google's homepage.

Timestamp API

Add a date, in any common format, or a unix timestamp to end of this URL and receive a response with a unix timestamp and the original date. The response is in JSON.

Examples:

{{request.build_absolute_uri}}/01-01-2001
{"natural": "January 1, 2001", "unix": "978307200"}
{{request.build_absolute_uri}}//1480896000
{"natural": "December 5, 2016", "unix": "1480896000"}
